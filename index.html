Project 2

주요 역할

  • 사업 입찰 서류 준비부터 기획, 개발, 배포까지 프로젝트 전반 참여
  • 대규모 비정형 문화 데이터 수집/관리/저장 플랫폼 설계 및 구현
  • 내부 관리 시스템(CMS) 개발 및 구축
  • 다양한 데이터 형식(Excel,3D 모델,이미지,동영상,음악 등)의 일괄 등록 기능 구현 (선배 개발자와 협업)
  • 동적 메뉴/카테고리 관리 및 게시판 생성 기능 개발
  • 사용자 친화적 JSP/Servlet 웹 화면 개발 및 반응형 디자인 적용

기술 스택

Java/Spring PostgreSQL JSP Servlet JavaScript/jQuery Linux Apache Tomcat

📌 핵심 성과: 대용량 데이터 쿼리 성능 최적화

안정화 기간 중 대용량 '문화백서' 데이터 조회 속도 저하 문제를 발견했습니다. 평소 학습했던 DB 인덱싱 개념을 떠올려 직접 구글링을 통해 PostgreSQL 테이블에 인덱스를 적용했고, 그 결과 조회 시간을 개선 전 25초에서 개선 후 4초 이내로 단축하는 데 성공했습니다. 이 경험은 단순 개발을 넘어 스스로 문제를 분석하고 해결하여 시스템 성능을 개선하는 역량을 키우는 계기가 되었습니다.

Project 3

한국문화정보원 문화 빅데이터 플랫폼 유지보수 추가개발 사업

주요 역할 및 성과

  • 매년 최신화되는 문화 빅데이터 수집 및 정제 작업
  • 데이터 생성부터 배포까지의 전체 프로세스 경험
  • Excel을 활용한 수작업 데이터 가공, 표준화 및 무결성 검증
  • 플랫폼에 배포 가능한 최종 데이터 제작 및 업로드

기술 스택

Excel Airflow

📌 핵심 성과: 데이터 핸들링 전체 프로세스 경험 및 자동화 이해

다양한 빅데이터를 다루는 과정에서 DB 쿼리뿐만 아니라 Excel을 활용한 수작업 데이터 가공 작업도 경험했습니다. 이를 통해 데이터가 생성되어 플랫폼에 배포되는 전체 프로세스를 깊이 이해하게 되었으며, Airflow를 이용한 자동 업로드 과정을 직접 보며 데이터 핸들링 업무 역량을 한 단계 더 성장시키는 계기가 되었습니다.

Project 4

Only 프론트엔드 개발 참여 및 경력 방향성 재정립

주요 역할 및 성과

  • 프론트엔드만 단독 개발하는 프로젝트를 맡아 Vue.js와 바닐라 자바스크립트를 주도적으로 학습하며 프로젝트 요구사항을 성공적으로 해결했습니다.
  • 위/아래 이동 및 클릭만으로 데이터의 정렬과 관리를 가능하게 하는 **재사용 가능한 컴포넌트**를 직접 구현하여 개발 생산성 향상에 기여했습니다.

기술 스택

Vue.js JavaScript HTML5 CSS3

📌 핵심 성과: 위기 상황에서의 적응력과 전문성 재확인

예상치 못한 프론트엔드 프로젝트에 투입되었지만, 이를 성장의 기회로 삼아 Vue.js와 바닐라 자바스크립트의 핵심 원리를 깊이 있게 학습했습니다. 특히, 위/아래 버튼과 클릭만으로 데이터 정렬 및 관리를 할 수 있는 **라이브러리 형태의 재사용 가능한 컴포넌트**를 직접 설계하고 구현하여 기술적 문제 해결 능력을 입증했습니다. 이 경험을 통해 낯선 기술 스택에 빠르게 적응하는 저의 능력을 확인했으며, 백엔드 개발자로서의 커리어 목표를 더욱 확고히 다지는 계기가 되었습니다.
